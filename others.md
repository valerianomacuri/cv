# Conseguir Trabajo como Frontend Developer

## Responsabilidades

* Pasar del diseño a la web usando Figma.
* Asegurar la compatibiliad en todos los dispositivos.
* Crear código reusable.
* Optimizar sitios web para mejorar la velocidad y escalabilidad.
* Hacer sitios web accesibles
* Crear interfaces de usuario usando React o Next.js
* Implementar autenticación en la aplicación web mediante email y Google usando REST API y firebase
* Usar librerias de UI para acelerar el desarrallo, librerias como Ant Design, Chakra UI, Material UI.
* Usar la metodología BEM para la escalabilidad del CSS.
* Maquetación web usando Flebox y Grid.
* Consumo de API's
